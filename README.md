Peer Instruction
================
   
# Dependencies

git -> http://git-scm.com/

node.js -> http://nodejs.org/

yeoman -> http://yeoman.io/

mongodb -> https://www.mongodb.org/
     
# Instalation

      git clone https://github.com/evertonrobertoauler/peer-instruction.git
      cd peer-instruction
      npm install
      bower install
 
# How to start the server

    node app.js 
    # or 
    grunt
    
and open the link: http://127.0.0.1:3000/
